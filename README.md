Install
Install the dependencies:

> npm install
> npm install -g sharp
> npm install -g pm2
Usage
run the Whatsapp bot
> npm start
or

> pm2 start main.js
> pm2 monit
stop the Whatsapp bot
for npm :

> ctrl + c
for pm2 :

> pm2 stop
after running it you need to scan the qr
