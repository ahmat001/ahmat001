This is a Bot Group Type, so you have to enter the bot into a group in order to fully use the command!

Install
Install the dependencies:

> npm install
> npm install -g sharp
> npm install -g pm2
Usage
run the Whatsapp bot
> npm start
or

> pm2 start main.js
> pm2 monit
stop the Whatsapp bot
for npm :

> ctrl + c
for pm2 :

> pm2 stop
